# Sound Signal GUI Application (MATLAB)

This repository contains a Digital Signal Processing (DSP) GUI application built entirely in MATLAB. The program allows users to generate native audio signals, add or remove noise, adjust audio properties like speed and volume, and filter or combine audio inputs interactively.

##  Application Features
- **Signal Generation:** Create pure beep tones and synthetic white noise waveforms.
- **Audio Manipulation:** Real-time controls to increase/decrease speed, volume, and apply basic audio echoes.
- **Signal Arithmetic:** Built-in functions to mathematically subtract or multiply different audio tracks.
- **Noise Control & Filtering:** Tools to artificially inject noise into a pristine clip and isolate/remove noise artifacts.
- **Visual Analysis:** Integrated charting function to generate a time-domain waveform plot of the audio signals.
